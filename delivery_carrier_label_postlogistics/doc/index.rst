#######################
Odoo Postlogistics WSBC
#######################

Contents:

Postlogistics WSBC is an Odoo_ module to generate delivery labels for
your packages. It uses `Postlogistics "Barcode" Webservice`_.

Odoo Postlogistics WSBC has been developped and is maintained by Camptocamp_.
:ref:`contributors`

*Learn how to* :ref:`contribute`


.. _Odoo: http://www.odoo.com
.. _`Postlogistics "Barcode" Webservice`: https://www.post.ch/en/business/a-z-of-subjects/dropping-off-mail-items/business-sending-letters/sending-consignments-web-service-barcode
.. _Camptocamp: http://www.camptocamp.com
.. _`source code is available on GitHub`: https://github.com/OCA/carrier-delivery
